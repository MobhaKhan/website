Develop a concise full-stack obituary creation app with React and AWS, seamlessly blending AI, cloud computing, and modern web development. Users can effortlessly generate obituaries with OpenAI GPT-3, convert them to speech using Amazon Polly, and store data on AWS DynamoDB. Serverless functionality is ensured with AWS Lambda, managed by Terraform. The React and Tailwind CSS front end provides a sleek interface. Note: Due to my AWS account connection, the application isn't deployed; but you can watch a demo video here. The code for this very cool app is available on GitHub .

It features a responsive design, seamless navigation with React Router, and a rich text editor powered by React Quill. This Wordle game, accessible at https://mskhan-wordle.netlify.app/, is a dynamic web app built with JavaScript, HTML, and CSS. It features a responsive grid where players guess the hidden word by filling in letters. The game includes interactive elements like instructions, hints, and a stylish dark mode. With a creative design and user-friendly interactions, this Wordle game delivers an enjoyable gaming experience. The code for this web-based app is available on GitHub .

This project seamlessly integrates Java and MySQL for a smart and efficient zoo management schedule. The code for this GUI-based app is available on GitHub .
